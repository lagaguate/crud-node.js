# Servidor WEB con Nodejs y Expresss

### Ejecutar proyecto
```npm index.js```

### Instalar varibles de entorno
``` npm i dotenv ```
### Instalar EJS, motor de vitas
``` npm i ejs ```
### Instalar todas las dependencias
``` shell
 npm i 
```

### Ejecutar proyecto
``` npm run dev ```
